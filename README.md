# Predictive Maintenance Dashboard  
**Avoiding Unplanned Downtime in Manufacturing Using Sensor Data**

![Project Banner]

## Project Overview

This portfolio project demonstrates an **end-to-end analytics solution** for predictive maintenance in a manufacturing environment (inspired by Industrie 4.0 concepts).  

The goal is to use real-time sensor data to **predict machine failures before they occur**, prioritize maintenance tasks, reduce unplanned downtime, and visualize potential cost savings.

### Key Features
- Risk scoring & early warning system based on sensor readings
- Heatmap of failure risks by machine type
- Breakdown of failure types (tool wear, heat dissipation, power, overstrain)
- KPI overview (failure rate, average wear, high-risk machines)
- What-If simulation for preventive actions (e.g., earlier tool change)
- Structured business analysis: requirements, stakeholder map, BPMN process models (As-Is vs To-Be)

### Business Value
- Simulated reduction of unplanned downtime: **20–35%**
- Estimated annual savings: **$150,000–$400,000** (for 8–12 critical machines)
- Shift from **reactive** to **predictive** maintenance
